## Gorilla Ecosystem

A self-hosted AI fitness coaching platform built entirely in Rust.

| Project | Description |

| [gorilla_coach](gorilla-coach.md) | Full-stack fitness app — periodized training, mesocycle management, dashboard, PWA (Leptos + Axum) |
